## Reference
 
 ![](./img/85.png)
- **\*** 表示 **pointer to ...**
- **&** 表示 **代表**，reference 内部也是通过指针实现

	```C++
    int& r = x; // x 必须被初始化，r 就无法被改变
    ```

- 测试程序

	![](./img/86.png)
    
- reference 常见用途

	![](./img/87.png)
	- 尝尝用作函数参数传入
	- 签名包含函数名与参数，但不包含返回类型

		```C++
        // 不能并存的情形
        double imag(const double& img) {...}
        double imag(const double img) {...}
        
        // 可以并存的情形
        double imag() const {...}
        double imag() {...}
        ```
